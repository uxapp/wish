سلام ، به پروژه متن باز آرزو خوش آمدید.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

اگه از نسخه تحت مک گیت هاب استفاده میکنید. فقط کافی است سینک کنید.

آرزو چیست ؟‌

آرزو یک کلاینت متن باز شبکه اجتماعی ای هست برای اشتراک گذاشتن آرزوهای شما.توسط این شبکه میتوانید آرزو کنید،برای آرزوی دیگران دعا یا کمک کنید و امتیاز کسب کنید تا آرزوهای ستاره دار بکنید.
